# Hotspot

![Preview](preview.png)

Plugin to create a Wi-Fi hotspot from your computer.

## Usage

- **Bar widget** — Shows hotspot status. Click to toggle on/off.
- **Right click** — Open plugin settings.

## Requirements

- **Noctalia Shell** ≥ 3.6.0
- **NetworkManager** with hotspot capability
- A Wi-Fi adapter that supports AP mode
